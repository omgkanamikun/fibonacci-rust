## General info
This project sows how to implement Fibonacci recursion-way sequences generator with cached values in mutable array.

## Technologies
Project is created with:
* Cargo version: 1.58.0 (f01b232bc 2022-01-19)
* rustc version: 1.58.1
* rustup version: 1.24.3 (ce5817a94 2021-05-31)

## Setup
To run this project, install it locally using npm:

```
$ cd ../fibonacci-rust
$ cargo run fibonacci-rust
```
